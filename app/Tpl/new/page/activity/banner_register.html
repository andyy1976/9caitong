    3. 2.可以在哪些地方使用玖财通？
    4. 玖财通APP端：可以在手机端各大应用市场安装“玖财通”使用。
      玖财通PC端：可在电脑浏览器中访问www.9caitong.com。
      玖财通WAP端：可在浏览器中访问wap.9caitong.com或关注玖财通微信公众号。
      玖财通微信端：可以在微信中搜索“玖财通”并关注该微信公众号。
    5. 3.在玖财通出借安全吗？
    6. 技术层面：
      平台采用全球先进的阿里云金融服务，国际标准的SSL安全加密技术，中国金融中心CFCA数字证书，在保证快速、高效交易的同时，最大限度保证用户交易安全。
      合规层面：
      玖财通自成立伊始，积极响应国家相关部门政策，在打造平台的同时，步步遵循合规指导意见。无论是ICP证书的获得，SSL 加密协议和CFCA数字证书的签订使用无一不体现这一点。目前正在洽谈银行存管系统，预计近期上线。

  • +实名认证
    1. 1.为何要实名验证？
    2. 为确认出借人的身份以及保证合同的有效性与合法性，玖财通需要对平台的所有用户进行身份验证，同时我们也会通过实名认证验证您设置的提现银行账户是否属于您本人。为了确保您的资金安全，您需要在首次出借前进行实名认证。当用户提现或维权时，平台都将核实身份信息。
  • +资金与回款
    1. 1.什么是募集期？出借后募集期间有收益吗？
    2. 根据互联网金融相关部门规定，网络借贷项目的资金募集期不超过20天。
      募集期是有收益的，募集期收益由玖财通平台补贴给用户作为站岗资金的福利。
      募集期收益计算方式为:出借金额*出借后募集天数*项目年化收益/365
    3. 2.出借后何时计息？如何派息？如何计算收益？
    4. 玖财通平台项目的还款方式为按月付息，到期还本。
      项目期间每期返还项目每期收益，最后一期返还最后一期收益和出借本金外，额外返还募集期收益。
      募集期收益=出借金额*出借后募集天数*项目年化收益/365
      项目收益=项目天数*出借金额*项目年化收益/365
  • +充值与提现
    1. 1.什么是同卡进出？为什么要选择同卡进出？
    2. 为保障用户资金安全，使用快捷支付的用户需遵循同卡进出的规则。同卡进出是指使用快捷支付的用户，绑定后充值和提现只能通过这一张银行卡来完成，即使手机丢失银行卡账号泄露资金仍不会提取到其他银行卡，最大限度保障用户的资金安全。
    3. 2.如何充值？
    4. 1)充值方式分为网银充值和快捷充值，暂不支持信用卡充值，每日的充值限额依据各银行限额为准。
      2)首次使用招商银行卡快捷充值，需要校验银行卡密码。后续再用招行卡充值不需要校验卡密。
      3)由于网络传输延迟问题，充值成功后可能玖财通账户可用余额未增加。如果遇到这种情况请您将付款成功的页面截图，然后刷新页面。如果刷新未能解决问题，请您拨打玖财通客服电话010-53608035，客服人员会马上处理您的问题。
    5. 3.提现资金多久到账？提现需要手续费吗？
    6. 提现申请成功后，将在1个工作日（双休日或法定节假日顺延）到账，请您注意查收。为保证提现额度与到账速度，请完善银行卡的支行信息。
      为防止恶意转账、套现等行为，充值的资金，未出借部分将按照提现金额收取0.5%的手续费，出借后的资金提现将不收取任何手续费。
  • +体验金与虚拟货币
    1. 1.什么是代金券和红包？如何使用？
    2. 1.代金券和红包是玖财通平台的虚拟货币，可用来出借使用。
      2.代金券使用比例为1:50，即出借1000元可使用20元代金券，红包没有使用比例限制；
      3.新手标和尾标不可使用红包/代金券；
      4.红包和代金券不可同时使用。
    3. 2.体验金怎样获得？如何使用？
    4. 体验金是玖财通发放给用户，用于体验出借的一种虚拟出借产品。完成注册即可分别获得8888注册体验金、58888出借体验金、16666分享体验金。
      注册体验金：可直接使用，计息结束后可领取收益。
      出借体验金：出借除新手标以外的任何标的后即可解锁使用，计息结束后可领取收益。
      分享体验金：邀请三个好友注册并实名认证，且好友均使用注册体验金后即可解锁使用，计息结束后可领取收益。
  • +邀请好友
    1. 1.如何邀请好友？
    2. 在玖财通APP【账户】-【邀请好友】页面通过以下三种方式均可邀请好友：
      1.点击“邀请好友”按钮，将邀请链接通过微信/朋友圈/QQ分享给好友，好友完成注册。
      2.好友扫描“我的二维码”进入到注册页面完成注册。
      3.好友在注册页面填写你的邀请码，完成注册。
      注册后30天内完成实名认证，即可成功邀请一位好友。
    3. 2.邀请好友有什么奖励？
    4. 每邀请一位好友您即可获得20元代金券，超过50人，不再发放奖励。
